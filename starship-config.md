# Starship config

## Create configuration file

Create `~/.config/starship.toml`

```sh
mkdir ~/.config 
touch ~/.config/starship.toml
```

## Customize prompt

Edit `~/.config/starship.toml`

```sh
helix ~/.config/starship.toml
```
