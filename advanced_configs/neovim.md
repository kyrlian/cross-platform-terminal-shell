# Neovim editor

https://neovim.io/

## Install 

### Macos

```sh
brew install neovim 
```
 	
### Windows

```sh
scoop install neovim
```

## Config file location

- Macos
    ```sh
    ~/.config/nvim/init.vim
    ```

- Windows
    ```sh
    ~/AppData/Local/nvim
    ```

## Display line number

Add to the config file 

```sh
set number
```
